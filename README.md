# react-antd-adminMT

> An out of the box enterprise level management system , build with react+antd+redux

### 本地开发

1. 安装依赖

```yarn install```

2. 本地运行

```yarn dev```

### 构建打包

1. 打包stg

```yarn build:stg```

2. 打包prod

```yarn build:prod```
